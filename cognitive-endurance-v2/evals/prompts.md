# Evaluation Prompts

Use these prompts to test whether the skill restores motion rather than just sounding supportive.

## 1. Ambiguous Ask

`老板说下周给他一个方案，我现在连方案是 PPT 还是文档都不知道，脑子很乱。`

Expected:

- route to `clarify`
- identify likely deliverable and unknowns
- optionally scaffold a clarification message

## 2. Deadline Panic

`今天下班前要交初稿，但内容还散着，我现在完全推不动。`

Expected:

- identify deadline compression
- define minimum shippable version
- give one visible next move

## 3. Reply Avoidance

`我需要回老板一个进度，但我迟迟不敢发。`

Expected:

- route to `communicate`
- provide a short status update draft

## 4. Blank Page

`我要写周报，但文档开着半小时了一个字都没写。`

Expected:

- route to `shape`
- provide a fillable shell, not generic encouragement

## 5. Stakeholder Dependency

`这事其实卡在产品那边不给信息，但我又不知道怎么催。`

Expected:

- identify missing input
- route to `communicate`
- draft a low-drama nudge

## 6. True Depletion

`算了，我真的不想干了，脑子已经没电了。`

Expected:

- route to `recover`
- stop task pressure
- provide one re-entry question only
