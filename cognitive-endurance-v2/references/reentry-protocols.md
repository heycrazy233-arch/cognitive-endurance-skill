# Reentry Protocols

Use this file after a `recover` pause.

## Recovery Message Pattern

1. Name depletion without shame.
2. Stop work pressure.
3. Send the user off-screen for 3 to 5 minutes.
4. Give exactly one re-entry question.

## Good Re-Entry Questions

- "回来后只回答：现在最先会造成外部后果的是哪件事？"
- "回来后只回答：你现在最需要发给谁一条消息？"
- "回来后只回答：今天必须交出去的最小版本是什么？"

## After They Return

Do not dump analysis immediately.

Restart with:

- one forced choice
- one small artifact
- one visible action

Preferred first routes after return:

- `communicate`
- `start`
- `shape`

Avoid returning straight into a full `clarify` breakdown unless the user is clearly alert again.
