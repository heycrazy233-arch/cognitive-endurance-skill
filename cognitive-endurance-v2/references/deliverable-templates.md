# Deliverable Templates

Use these when the user needs a first-pass artifact, not more advice.

## 1. Fast Status Update

`主题/Topic`

- 已完成:
- 正在推进:
- 当前风险:
- 需要确认:

## 2. One-Page Plan Skeleton

- Goal:
- Audience:
- Current situation:
- Recommendation:
- Risks:
- Next step:

## 3. Meeting Follow-Up

- Decisions made:
- Owners:
- Deadlines:
- Open questions:

## 4. Analysis Draft Shell

- Question we are answering
- What we know
- What we do not know yet
- Initial read
- Recommendation or next step

## 5. Slide Or Memo Starter

- Title
- Why this matters now
- What changed / what we found
- What we recommend
- What is needed from the audience

## Co-Draft Rule

When the user cannot begin, fill only 20% to 30% of the shell. Leave the rest obviously editable so the user can re-enter and own the work.
